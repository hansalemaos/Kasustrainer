# Kasustrainer [DOWNLOAD THE INSTALLER HERE](https://www.dropbox.com/s/ytccdrp97wfe4pa/kasustrainer_setup.exe?dl=0)



## What can you do with it?

- Practice the German cases: Nominativ, Akkusativ, Dativ, Genitiv
- Use your own text to practice
- Multiple choice questions
- No coding required!
- Download,install and start
- Source code included!
- For Windows!
